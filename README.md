# CodeAlpha Data Science Internship

This repository contains projects completed as part of my Data Science Internship at CodeAlpha.

Task 1: Iris Flower Classification

- Built classification models using Decision Tree and Naive Bayes
- Performed data preprocessing and train-test split
- Evaluated model performance using accuracy and confusion matrix
- Achieved high accuracy due to well-structured dataset


Task 2: Unemployment Analysis

- Performed data cleaning and preprocessing
- Analyzed unemployment trends over time
- Studied the impact of COVID-19 on unemployment rates
- Identified seasonal patterns (higher unemployment in April–May)
- Visualized insights using graphs


Tools & Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

Key Insights

- The Iris dataset is highly structured, allowing models like Decision Tree and Naive Bayes to achieve very high accuracy
- Different machine learning models can perform similarly well on simple datasets
- Unemployment rate spiked significantly during the COVID-19 lockdown period
- Gradual recovery in employment was observed after mid-2020
- Seasonal trends indicate higher unemployment in April–May
